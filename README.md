# MyVacations

Planifica, gestiona y controla vacaciones, presupuestos y gastos de viaje con Kotlin Multiplatform y Compose Multiplatform.

---

# 🇪🇸 Español

## Descripción

MyVacations es una aplicación desarrollada con Kotlin Multiplatform que permite planificar viajes, gestionar presupuestos, registrar gastos y analizar costes de vacaciones desde una única base de código compartida para Android e iOS.

El proyecto nace como una aplicación real y como una demostración de buenas prácticas utilizando tecnologías modernas del ecosistema Kotlin.

## Funcionalidades

### Dashboard

* Próximos viajes
* Resumen general
* Presupuesto total
* Gastos acumulados
* Accesos rápidos

### Gestión de vacaciones

* Crear viajes
* Editar viajes
* Eliminar viajes
* Consultar detalles
* Organizar viajes por fechas

### Control de presupuesto

* Definir presupuesto por viaje
* Registrar gastos
* Calcular presupuesto restante
* Detectar sobrecostes

### Gestión de gastos

Categorías predeterminadas:

* ✈️ Transporte
* 🏨 Alojamiento
* 🎯 Actividades

Categorías personalizadas:

* Crear categorías propias
* Seleccionar icono
* Asignar gastos específicos

### Estadísticas

* Gasto total
* Coste por viajero
* Coste por día
* Gastos por categoría
* Historial de viajes

# 🇬🇧 English

## Description

MyVacations is a Kotlin Multiplatform application designed to help users plan trips, manage budgets, track expenses and analyze vacation costs using a shared codebase for Android and iOS.

The project serves both as a real-world application and as a showcase of modern Kotlin Multiplatform development practices.

## Features

### Dashboard

* Upcoming trips
* Travel overview
* Budget summary
* Expense tracking
* Quick actions

### Vacation Management

* Create vacations
* Edit vacations
* Delete vacations
* View details
* Organize trips by date

### Budget Tracking

* Define a budget per trip
* Track expenses
* Calculate remaining budget
* Detect budget overruns

### Expense Management

Default categories:

* ✈️ Transport
* 🏨 Accommodation
* 🎯 Activities

Custom categories:

* Create custom categories
* Select icons
* Track specific expenses

### Analytics

* Total spending
* Cost per traveler
* Cost per day
* Spending by category
* Travel history insights
